# Shrimp’s Job Backend

ENGLISH
Shrimp’s Job is a community-driven web platform designed for small businesses and local neighborhoods.
It allows users to publish services or job requests and enables verified users to take those jobs or promote their own offerings.
The goal of the project is to improve visibility between people who need help with specific tasks and others in the same area who can provide those services.

ESPAÑOL
Shrimp’s Job es una plataforma web enfocada en comunidades y pequeños negocios locales.
Permite a los usuarios publicar solicitudes de trabajos o servicios y, a usuarios verificados, aceptar esas solicitudes o promocionar sus propios servicios.
El objetivo del proyecto es mejorar la conexión entre personas que necesitan ayuda con tareas específicas y otras dentro del mismo vecindario que pueden ofrecer ese apoyo.

Backend API for a job platform built with Java and Spring Boot.

## Tech Stack
- Java 17
- Spring Boot 3
- Maven
- JPA / Hibernate
- MySQL
- Git

## Current Features
- User module (CRUD)
- Role entity
- REST controllers
- Database integration

## Run Locally

1. Clone repository
2. Configure database credentials using environment variables
3. Run:

mvn spring-boot:run

Server runs on: http://localhost:8081
