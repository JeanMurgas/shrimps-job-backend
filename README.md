# Shrimp’s Job Backend

Backend API for a job platform built with Java and Spring Boot.

## Tech Stack
- Java 17
- Spring Boot 3
- Maven
- JPA / Hibernate
- MySQL
- Git

## Current Features
- User module (CRUD)
- Role entity
- REST controllers
- Database integration

## Run Locally

1. Clone repository
2. Configure database credentials using environment variables
3. Run:

mvn spring-boot:run

Server runs on: http://localhost:8081
